# 项目概要

## 项目名称

英文仓库名：`boss-jobpilot`

中文工作名：AI 求职代理

## 背景

用户主要在 Boss 直聘等招聘平台投递岗位。痛点包括岗位筛选慢、公司背景判断慢、打招呼语重复、简历需要针对不同岗位改写、投递后跟进混乱、面试准备分散。

本项目目标是将这些重复流程整合为一个本地优先的 AI 工具，让用户将完整经历素材交给系统，由系统根据岗位和公司要求自动生成精准投递包。

## 核心目标

- 建立结构化个人经历库，而不是只维护一份基础简历。
- 自动采集当前岗位和公司信息。
- AI 解析岗位要求、公司偏好和风险信号。
- 自动匹配经历素材，生成个性化简历。
- 自动生成 Boss 打招呼语、跟进语、自我介绍和面试准备材料。
- 管理投递状态和后续跟进。
- 统计不同岗位、简历版本和话术的效果。

## 自动化等级

| 等级 | 描述 | MVP 支持 |
| --- | --- | --- |
| L1 | 只分析，不操作 | 是 |
| L2 | 生成简历和话术，用户手动发送 | 是 |
| L3 | 自动填入内容，用户确认发送 | 是 |
| L4 | 符合规则的岗位自动打招呼 | 后续 |
| L5 | 自动投递、自动跟进、自动复盘 | 后续谨慎开放 |

## 非目标

- 不做绕过平台风控的批量攻击式自动化。
- 不编造用户经历、项目数据或工作职责。
- 不将用户简历、投递记录默认上传到第三方自有服务。
- 不承诺替代用户判断面试邀约和职业选择。
